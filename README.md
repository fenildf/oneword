# 随记单词 for iOS
[![Build Status](https://travis-ci.org/woodyyan/oneword.svg?branch=master)](https://travis-ci.org/woodyyan/oneword)

我们一天会解锁手机100多次
每次解锁，在通知栏停留那几秒，足够记忆一个单词

根据我们自己的真实经历，边读边写，是记忆单词最高效的方式之一

随记单词
希望帮你在查看手机通知的同时，轻松记住单词
在手机上，还原真实直接的单词记写体验
